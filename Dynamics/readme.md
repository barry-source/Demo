iOS不能做到禁止截屏和录屏，只能使用通知监听到截屏或录屏事件，来做一些处理，比如停止播放视频，视频加盖蒙板等。

截图通知名：
```
    UIApplication.userDidTakeScreenshotNotification
```
录屏通知名：
-  录屏功能iOS11之后才有
```
    UIApplication.capturedDidChangeNotification
```
